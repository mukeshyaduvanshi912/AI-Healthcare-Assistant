# 🏥 AI Healthcare Assistant

An AI-powered healthcare assistant that predicts diseases based on symptoms using Machine Learning.

## 🚀 Live Demo
https://ai-healthcare-assistant-e66r.vercel.app

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js + Express
- AI Service: Flask (Python)
- Database: MongoDB

## 📂 Project Structure
- frontend/ → React app
- backend/ → Node.js API
- ai-service/ → Flask ML model

## ✨ Features
- Symptom-based disease prediction
- AI-powered diagnosis
- Doctor & patient dashboard
- Secure authentication

## ⚙️ How to Run Locally
```bash
# frontend
cd frontend
npm install
npm start

# backend
cd backend
npm install
npm run dev

# AI service
cd ai-service
pip install -r requirements.txt
python app.py
